# user-management-app

### Setup
- git clone https://github.com/vyatsenk0/user-management-app.git
- cd user-management-app
- npm install

### Run
- npm start / npm start lite

## Implemented features
- React + TypeScript, React Router (user list + add/edit form), users table with ID, Name, Email, Skills, Registration date
- Add/edit/delete users via React Hook Form with dynamic skills, prefilled edit form, and validation (required fields, email format)
- Pagination and sorting in table, UI with Material UI
- Git repository with versioned commits
- Data persistence via localStorage; JSONPlaceholder API initially used during development
